# Privacy Policy for Flavortown Utils

**Last Updated:** December 23, 2025

## Overview

Flavortown Utils is a browser extension that enhances the user experience on flavortown.hackclub.com with custom themes and UI improvements.

## Data Collection

**We do not collect, store, or transmit any personal data.**

### What We Store Locally

The extension stores the following data **locally on your device only** using the browser's storage API:

- **Theme preference** (which theme you selected)
- **Custom color settings** (if using custom theme)
- **Sidebar pin state** (whether sidebar is pinned)

This data:
- Never leaves your device
- Is not transmitted to any server
- Is not shared with any third party
- Can be cleared by uninstalling the extension

## Permissions Explained

| Permission | Why We Need It |
|------------|----------------|
| `storage` | Save your theme preferences locally |
| `tabs` | Apply theme changes to open Flavortown tabs in real-time |
| `Host (flavortown.hackclub.com)` | Inject CSS themes on the Flavortown website |

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking.

## Changes to This Policy

Any changes to this privacy policy will be reflected in this document with an updated date.
